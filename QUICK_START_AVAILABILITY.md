# Quick Start Guide - Availability Confirmation

## For Partners: How to Send Availability Confirmation

### Step 1: Accept and Schedule a Pickup
1. Go to **Assigned Waste Requests** page
2. Find a waste request with status "Assigned" or "Requested"
3. Click **Accept** button
4. Click **Schedule Pickup** button
5. Fill in the form:
   - Select method: Pickup or Drop-off
   - Choose date (must be future date)
   - Select time slot
6. Click **Schedule Collection**

### Step 2: Send Availability Confirmation
1. After scheduling, you'll see a new button: **🔔 Confirm Availability**
2. Click the button
3. System will check timing:
   - ✅ If within 12 hours: Sends immediately
   - ⚠️ If more than 12 hours: Shows warning but allows sending
4. Confirmation sent to user!

### Step 3: Check User Response
1. Go to **Notifications** page
2. Look for responses from users
3. Status will show:
   - ✅ **Confirmed**: User is available
   - ❌ **Declined**: User is not available (reschedule needed)
   - ⏳ **Pending**: Waiting for user response

---

## For Users: How to Respond to Availability Confirmation

### Step 1: Receive Notification
1. You'll receive a notification with:
   - Partner name
   - Pickup date and time
   - Request for confirmation

### Step 2: Review Details
1. Check the pickup details:
   - Date
   - Time slot
   - Location
2. Verify you'll be available

### Step 3: Respond
1. Click one of two buttons:
   - **✓ Confirm**: If you'll be available
   - **✗ Decline**: If you won't be available
2. Your response is saved immediately
3. Partner is notified of your response

---

## Best Practices

### For Partners:
- ⏰ Send confirmations 12 hours before pickup
- 📞 Follow up if no response within 2 hours
- 🔄 Be ready to reschedule if user declines
- 📝 Keep notes on user preferences

### For Users:
- ⚡ Respond quickly (within 1-2 hours)
- 📱 Enable notifications for faster alerts
- 🗓️ Check your calendar before confirming
- 💬 Contact partner if you need to reschedule

---

## Troubleshooting

### Partner: "Confirm Availability button not showing"
**Solution**: Make sure you've scheduled the pickup first. Button only appears after scheduling.

### Partner: "User not responding"
**Solutions**:
1. Wait 2 hours for response
2. Call user directly using phone number
3. Send another reminder
4. Proceed with pickup if urgent

### User: "Didn't receive notification"
**Solutions**:
1. Check Notifications page in app
2. Refresh the page
3. Check notification settings
4. Contact partner directly

### User: "Need to change time after confirming"
**Solutions**:
1. Contact partner immediately
2. Explain situation
3. Partner can reschedule
4. Confirm new time

## Example Scenarios

### Scenario 1: Perfect Flow ✅
```
Partner: Schedules pickup for Jan 16, 9 AM
Partner: Sends confirmation on Jan 15, 9 PM (12 hours before)
User: Receives notification immediately
User: Confirms within 30 minutes

```

```
Partner: Sends confirmation

Partner: Receives decline notification
Partner: Contacts user to reschedule

Partner: Sends new confirmation
User: Confirms new time
Result: Rescheduled successfully! 🔄
```

### Scenario 3: No Response ⏳
```
Partner: Sends confirmation
User: No response after 2 hours
Partner: Calls user directly
User: Confirms over phone
Partner: Proceeds with pickup
Result: Pickup completed! ✅
```
